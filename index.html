Git əmrləri

git konfiqurasiyası

İstifadəsi:git config –global user.name “[name]”  

İstifadəsi:git config –global user.email “[email address]”  

Bu əmr öhdəliklərinizlə istifadə olunmaq üçün müvafiq olaraq müəllif adını və e-poçt ünvanını təyin edir.

git init

İstifadəsi:git init [repository name]

 

Bu əmr yeni repozitoriya başlamaq üçün istifadə olunur.

git klonu

İstifadəsi:git clone [url]  

Bu əmr mövcud URL-dən depo əldə etmək üçün istifadə olunur.

git əlavə et

İstifadəsi:git add [file]  

Bu əmr səhnələşdirmə sahəsinə fayl əlavə edir.

İstifadəsi:git add *  

Bu əmr səhnələşdirmə sahəsinə bir və ya daha çox əlavə edir.

git commit

İstifadəsi:git commit -m “[ Type in the commit message]”  

Bu əmr faylı daimi olaraq versiya tarixçəsində qeyd edir və ya snapshot edir.

İstifadəsi:git commit -a  

Bu əmr git add əmri ilə əlavə etdiyiniz hər hansı faylları və həmçinin o vaxtdan bəri dəyişdirdiyiniz faylları icra edir.

git fərq

İstifadəsi:git diff  

Bu əmr hələ səhnələşdirilməmiş fayl fərqlərini göstərir.

 Usage: git diff –staged 

Bu əmr səhnələşdirmə sahəsindəki fayllarla mövcud olan ən son versiya arasındakı fərqləri göstərir.

İstifadəsi:git diff [first branch] [second branch]  

Bu əmr qeyd olunan iki filial arasındakı fərqləri göstərir.

git sıfırla

İstifadəsi:git reset [file]  

Bu əmr faylı ləğv edir, lakin faylın məzmununu qoruyur.

İstifadəsi:git reset [commit]  

Bu əmr göstərilən öhdəlikdən sonra bütün öhdəlikləri ləğv edir və dəyişiklikləri yerli olaraq saxlayır.

İstifadə: git reset –hard [commit] Bu əmr bütün tarixçəni ləğv edir və müəyyən edilmiş öhdəliyə qayıdır.

Git sirlərini Jenkins boru kəməri ilə necə birləşdirəcəyini öyrənin .

git statusu

İstifadəsi:git status  

Bu əmr icra edilməli olan bütün faylları sadalayır.

git rm

İstifadəsi:git rm [file]  

Bu əmr faylı iş kataloqunuzdan silir və silinməni mərhələlərlə həyata keçirir.

git log

İstifadəsi:git log  

Bu əmr cari filial üçün versiya tarixçəsini siyahıya almaq üçün istifadə olunur.

İstifadəsi:git log –follow[file]  

Bu əmr faylların adlarının dəyişdirilməsi də daxil olmaqla, fayl üçün versiya tarixçəsini sadalayır.

git şou

İstifadəsi:git show [commit]  

Bu əmr müəyyən edilmiş öhdəliyin metadata və məzmun dəyişikliklərini göstərir.

git tag

İstifadəsi:git tag [commitID]  

Bu əmr müəyyən edilmiş öhdəliyə teqlər vermək üçün istifadə olunur.

git filialı

İstifadəsi:git branch  

Bu əmr cari depodakı bütün yerli filialların siyahısını verir.

İstifadəsi:git branch [branch name]  

Bu əmr yeni filial yaradır.

İstifadəsi:git branch -d [branch name]  

Bu əmr xüsusiyyət bölməsini silir.

git checkout

İstifadəsi:git checkout [branch name]  

Bu əmr bir filialdan digərinə keçid üçün istifadə olunur.

İstifadəsi:git checkout -b [branch name]  

Bu əmr yeni filial yaradır və ona keçid edir.

git birləşmə

İstifadəsi:git merge [branch name]  

Bu əmr göstərilən filialın tarixçəsini cari filiala birləşdirir.

git uzaqdan

İstifadəsi:git remote add [variable name] [Remote Server Link]  

Bu əmr yerli anbarınızı uzaq serverə qoşmaq üçün istifadə olunur.

git push

İstifadəsi:git push [variable name] master  

Bu əmr master filialda edilən dəyişiklikləri uzaq deponuza göndərir.

İstifadəsi:git push [variable name] [branch]  

Bu əmr filial öhdəliklərini uzaq deponuza göndərir.

İstifadəsi:git push –all [variable name]  

Bu əmr bütün filialları uzaq deponuza itələyir.

İstifadəsi:git push [variable name] :[branch name]  

Bu əmr uzaq deponuzdakı filialı silir.

git çəkin

İstifadəsi:git pull [Repository Link]  

Bu əmr uzaq serverdəki dəyişiklikləri iş kataloqunuza gətirir və birləşdirir.

git stash

İstifadəsi:git stash save  

Bu əmr bütün dəyişdirilmiş izlənilən faylları müvəqqəti olaraq saxlayır.

İstifadəsi:git stash pop  

Bu əmr ən son gizlənmiş faylları bərpa edir.

İstifadəsi:git stash list  

Bu əmr bütün gizli dəyişiklikləri siyahıya alır.

İstifadəsi:git stash drop  

Bu əmr ən son gizlədilən dəyişiklikləri silir.
